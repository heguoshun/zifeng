# 离线分析

## 功能定位

回答「当前有多少离线设备、离线集中在哪些维度」。

## 展示结构

| 区域 | 说明 |
| --- | --- |
| 标题 | 「离线分析」+ 分布说明 +「查看设备」 |
| 环形图 | 中心：离线设备总数 |
| 明细列表 | 各分段离线数量与占比；点击跳转设备管理 |

不展示独立图例、**不展示**离线时长分布。

## 分布维度说明（标题副文案）

| 产品筛选 | 说明文案 |
| --- | --- |
| 全部 / 水务设备(water) | 按产品类型统计离线设备分布 |
| 有子分类的分类 | 按子分类统计离线设备分布 |
| 具体产品 | 按分组统计离线设备分布 |
| 其他分类 | 按产品统计离线设备分布 |

## 统计规则

- 只统计 **状态 = 离线** 的设备；故障不计入。
- 当前快照，切换时间范围不变。
- 不得用「总量 − 在线数」推算。
