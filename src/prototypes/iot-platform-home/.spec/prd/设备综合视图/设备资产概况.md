# 设备资产概况

## 功能定位

回答「设备分布在哪些产品或分组上」。

## 展示结构

| 区域 | 说明 |
| --- | --- |
| 标题 | 「设备资产概况」+ 分布说明文案 |
| 维度切换 | 仅当 **产品 = 全部** 时展示「按产品 / 按分组」切换 |
| 环形图 | 中心：设备总数；标签：{维度}分布 |
| 明细列表 | 各分段数量、占比；点击跳转设备管理 |

## 标题说明文案

| 条件 | 文案 |
| --- | --- |
| 产品 = 全部 且 按产品 | 共 N 台设备，按产品类型分布 |
| 产品 = 全部 且 按分组 | 共 N 台设备，按分组类型分布 |
| 产品 ≠ 全部 | 共 N 台设备，按分组类型分布（隐藏维度切换） |

## 统计规则

- 当前产品筛选 + 数据权限内已建档设备。
- 不受时间范围影响（快照）。
- 分类展示中文名称，禁止展示分类 ID 或拼音。
