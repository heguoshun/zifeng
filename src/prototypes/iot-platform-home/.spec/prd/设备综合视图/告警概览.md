# 告警概览

## 功能定位

「告警与工单」区域左侧，展示统计窗口内告警状态分布。

## 展示结构

| 区域 | 说明 |
| --- | --- |
| 共用标题 | 「告警与工单」+ 副文案「按状态查看当前告警与工单分布」 |
| 告警侧按钮 | 「查看告警」 |
| 告警环形图 | 中心：告警总量；分段：未处理 / 处理中 / 已处理 |
| 状态指标行 | 各状态数量与占比 |

**不展示**：告警总量环比、最高告警等级、独立风险提示条。

## 指标口径

| 指标 | 说明 |
| --- | --- |
| 告警总量 | 统计窗口内触发的告警总数 = 未处理 + 处理中 + 已处理 |
| 告警处理率 | 展示在概况区卡片，非本区单独成卡 |
| 概况卡片摘要 | 「当前待处理告警 N **条**」—— **当前未处理告警快照** |
| 环形图「未处理」 | **统计窗口内**触发的未处理告警分段 |

**注意**：概况卡片「当前待处理告警」与环形图「未处理」口径不同，不要求数值相等。

## 交互

- 点击「查看告警」或告警环形图区域 → 跳转设备告警信息。
- 点击概况区「当前待处理告警 N 条」→ 打开「待处理告警设备」抽屉。
- **原型现状**：跳转不带筛选参数。
