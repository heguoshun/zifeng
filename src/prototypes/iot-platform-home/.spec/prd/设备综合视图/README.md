# 设备总览

## 文档说明

本文档以当前可运行原型 `ExecutiveDeviceOverview` 为准编写。统计口径类细节见同级「指标计算口径」；若原型尚未实现的能力（如跳转带参、权限隐藏），在文中单独标注为**目标态**。

## 功能定位

设备综合视图是设备接入模块的默认首页，用于让管理人员在进入平台后第一时间判断设备资产规模、在线健康、数据接收质量、告警压力和工单闭环情况。

## 页面组成

| 区域 | 展示内容 | 交互目的 |
| --- | --- | --- |
| 页头 | 标题、整体状态标识、数据更新时间、刷新按钮 | 确认页面身份与数据新鲜度。 |
| 顶部筛选区 | 时间范围、产品树、指标口径 | 控制全页统计口径。 |
| 设备运行概况 | 设备资产（总量、在用/闲置）+ 设备在线率 / 数据正常率 / 告警处理率 | 快速判断资产规模与健康风险。 |
| 设备资产概况 | 环形占比 + 明细列表 | 判断资产结构。 |
| 离线分析 | 当前离线设备分布环形图 + 明细列表 | 判断离线问题集中位置。 |
| 数据接收异常 | 当前异常设备数、较上期变化、趋势条、异常类型、正常率徽标 | 判断数据链路质量。 |
| 告警与工单 | 告警三态环形图 + 工单四态环形图 | 判断异常处置闭环。 |
| 设备接入流程 | 五步接入导航 | 引导完成接入配置。 |

## 顶部筛选字段

| 字段/控件 | 类型 | 默认值 | 说明 |
| --- | --- | --- | --- |
| 时间范围 | 分段按钮 | 近7天 | 选项：今日、近7天、近30天。 |
| 产品 | 树形选择 | 全部 | 占位文案「全部」；可选全部、产品分类或具体产品。 |
| 刷新 | 按钮 | — | 位于页头；点击后文案变为「更新中」，完成后更新「数据更新于 …」。 |
| 数据更新时间 | 文本 | — | 页头展示「数据更新于 {时间}」。 |
| 指标口径 | 文本按钮 | — | 打开浮层，展示简化口径说明（非完整指标文档）。 |

## 筛选联动规则

1. 默认：近7天 + 产品 = 全部。
2. 切换时间范围：在线率、数据正常率、告警处理率及环比、数据接收异常趋势图同步刷新；设备总量、当前在线/离线台数、当前异常设备数、当前待处理告警条数等快照不变。
3. 切换产品：资产、离线、异常、告警、工单按新范围刷新。
4. 选择产品分类：统计该分类及下级包含的全部产品设备。
5. 产品 ≠ 全部 时：设备资产概况隐藏「按产品/按分组」切换，固定按分组展示。
6. 刷新不改变筛选，仅重拉当前条件下数据。

## 健康状态规则

| 整体状态 | 判定规则 | 页头展示文案 |
| --- | --- | --- |
| 正常 | 三项核心指标均达正常标准 | 正常 |
| 关注 | 任一项处于关注区间，且无异常 | 关注 |
| 异常 | 任一项处于异常区间 | 异常 |

单项指标卡右上角状态标签同为：**正常 / 关注 / 异常**（不加「运行」「需要」「存在」等修饰）。悬浮/聚焦展示阈值图与当前值。

**原型现状**：页头整体状态暂固定展示「正常」；正式实现应按上表随指标动态切换。

## 核心指标卡

| 指标 | 页面展示 | 判断标准 | 下钻 |
| --- | --- | --- | --- |
| 设备总量 | 总数 + 较上一周期变化 | 不做健康判断 | 无 |
| 在用设备 | 数量 + 占比 | — | 点击「在用设备」区域 |
| 闲置设备 | 数量 + 占比 | — | 点击「闲置设备」区域 |
| 设备在线率 | 百分比 + 环比 + 状态标签 | 正常 ≥98%；关注 95%～98%；异常 <95% | 点击「当前在线 N 台」「当前离线 N 台」 |
| 数据正常率 | 百分比 + 环比 + 状态标签 | 正常 ≥99%；关注 97%～99%；异常 <97% | 点击「当前异常设备 N 台」 |
| 告警处理率 | 百分比 + 环比 + 状态标签 | 正常 ≥95%；关注 85%～95%；异常 <85% | 点击「当前待处理告警 N 条」 |

指标卡本体**不展示**进度条、跳转箭头。完整计算公式见「指标计算口径」。

## 指标口径浮层（页面实际文案）

浮层为简化说明，与完整口径文档不完全等价：

| 指标 | 浮层说明 |
| --- | --- |
| 设备总量 | 当前已建档的全部设备数量。 |
| 设备在线率 | 在线设备数 ÷ 设备总量 × 100%。 |
| 数据正常率 | 正常接收并解析的数据条数 ÷ 应上报数据条数 × 100%。 |
| 告警处理率 | 已处理告警数 ÷ 告警总数 × 100%。 |

补充说明：在线率、数据正常率、告警处理率及环比随时间范围变化；设备总量、当前在线/离线台数、当前异常设备、当前待处理告警为快照，切换时间范围不变；数据接收异常趋势图按所选周期展示历史波动。

## 设备状态口径

四态互斥：**在线、离线、故障、禁用**。故障 ≠ 离线；不得用「总量 − 在线数」推离线数。详见「指标计算口径」。

## 空态与异常

| 场景 | 处理 |
| --- | --- |
| 当前范围无设备 | 指标为 0，图表空态 |
| 接口失败 | 对应区域失败态 + 重试 |
| 权限不足 | **目标态**：隐藏无权限入口 |

1. 默认近7天 + 产品全部。

3. 抽屉入口文案、标题、表头字段与上表一致。
4. PRD 不写本页未实现的图表与跳转带参能力。

# 数据接收异常

## 功能定位

回答「此刻有多少设备数据异常、近期如何波动、异常类型分布」。

## 展示结构

| 区域 | 说明 |
| --- | --- |
| 标题 | 「数据接收异常」 |
| 副说明 | 「当前异常设备快照；趋势按 {今日逐小时/近七日逐日/近三十日逐日} 展示历史波动」 |
| 徽标 | 「正常率 {百分比}%」 |
| 左侧 | 当前异常设备数、较上期变化（如「较前7天增加 12 台」）、趋势条 |

## 指标规则

| 指标 | 随时间范围变化 |
| --- | --- |
| 当前异常设备数 | 否（快照） |
| 异常类型分布 | 否（按当前仍异常设备拆分） |
| 正常率徽标 | 是（与概况区数据正常率同源） |
| 趋势条 | 是 |

# 产品接入概览

页面标题为 **「设备接入流程」**。

## 功能定位

五步接入路径导航，不参与时间范围筛选。

## 接入步骤（页面实际）

| 步骤 | 节点 | 说明 | 跳转 |
| --- | --- | --- | --- |
| 1 | 创建产品 | 定义设备类型与通信参数 | 产品管理 |
| 2 | 定义物模型 | 配置属性、事件与服务 | 产品管理 |
| 3 | 配置接入 | 设置协议、证书与网络服务 | 协议管理（主按钮）；子标签：协议管理、证书管理、网络服务 |
| 4 | 添加设备 | 注册设备并绑定产品 | 设备管理 |
| 5 | 设备上线 | 验证连接与数据上报 | 设备管理 |

## 展示规则

1. 横向步骤 + 箭头连接。
2. 第 3 步展示三个子入口标签，点击子标签独立跳转，不触发步骤主按钮。
3. **不展示**步骤完成态（已完成/待配置）。
4. **原型现状**：跳转不带产品筛选参数；未接入权限置灰。

**目标态**（非原型现状）：按产品范围判断完成态；无权限置灰并提示。

# 指标计算口径

完整统计口径文档，供后端接口与验收使用。**页面 UI 未必全部展示**——未展示的指标在表中标注「本页不展示」。

### 通用统计前提

| 项目 | 规则 |
| --- | --- |
| 统计范围 | 当前用户数据权限 + 顶部「产品」筛选 |
| 禁用/未启用设备 | 不参与在线率、数据正常率分母 |
| 故障设备 | 计入设备总量；不算在线、不算离线 |
| 环比 | 今日→较昨日；近7天→较前7天；近30天→较前30天 |
