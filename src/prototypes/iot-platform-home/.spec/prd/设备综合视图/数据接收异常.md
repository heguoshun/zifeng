# 数据接收异常

## 功能定位

回答「此刻有多少设备数据异常、近期如何波动、异常类型分布」。

## 展示结构

| 区域 | 说明 |
| --- | --- |
| 标题 | 「数据接收异常」 |
| 副说明 | 「当前异常设备快照；趋势按 {今日逐小时/近七日逐日/近三十日逐日} 展示历史波动」 |
| 徽标 | 「正常率 {百分比}%」 |
| 左侧 | 当前异常设备数、较上期变化（如「较前7天增加 12 台」）、趋势条 |
| 右侧 | 解析失败、缺报、非整点上报 的数量与占比 |

## 指标规则

| 指标 | 随时间范围变化 |
| --- | --- |
| 当前异常设备数 | 否（快照） |
| 异常类型分布 | 否（按当前仍异常设备拆分） |
| 正常率徽标 | 是（与概况区数据正常率同源） |
| 趋势条 | 是 |

异常类型行**不可**跨页跳转。
