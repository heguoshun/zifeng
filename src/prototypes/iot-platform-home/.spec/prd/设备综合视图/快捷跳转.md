# 快捷跳转

## 跳转入口（与原型一致）

| 入口 | 触发位置 | 目标 | 原型是否带参 |
| --- | --- | --- | --- |
| 明细抽屉 | 概况区 6 个下钻入口 | 当前页抽屉 | 保留当前页筛选 |
| 查看设备 | 离线分析按钮、资产/离线列表项 | 设备管理 | 否 |
| 查看告警 | 告警与工单标题按钮、告警环形图 | 设备告警信息 | 否 |
| 查看工单 | 工单环形图 | 工单管理 | 否 |
| 接入流程 | 五步及第 3 步子入口 | 对应管理页 | 否 |
| 指标口径 | 筛选栏按钮 | 当前页浮层 | — |

## 目标态（正式产品）

跨页跳转应带入可识别的产品、时间范围、设备状态或告警/工单状态；无权限时隐藏或置灰入口。

## 交互

- 可点击元素需有 hover/焦点反馈；支持 Enter/空格。
- 刷新 →「更新中」→ 更新「数据更新于」。
- 指标口径浮层：点击关闭按钮或浮层外关闭。
