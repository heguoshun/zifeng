# 大表数据分析页决策快照

## 产品需求

- 数据监测分析列表移除“设备详情”和“指标分析”，合并为“数据分析”。
- 点击“数据分析”进入独立页面，展示当前大表的用水和报警分析。
- 核心指标：日用水、月用水、时用水、夜间用水、报警数据。
- 日、月、时用水以趋势图展示；夜间用水按周期展示统计摘要和峰谷时刻分布。

## 设计决策

- 沿用大表中心现有导航、面包屑、白色面板和按钮样式。
- 顶部展示设备身份、区域、用户、状态和最后数据时间。
- 设备信息下方通过页签切换历史数据、日用水、月用水、时用水、夜间用水和报警数据。
- 夜间用水默认展示本月，可切换昨日、本月、本年或自定义日期时间范围。
- 夜间用水使用三张相互独立且样式一致的统计卡，展示累计夜间用水、峰值流量、谷值流量及峰谷发生日期时刻。
- 夜间用水不展示逐日明细；下方以柱状图展示峰谷时刻分布，横坐标只覆盖配置的夜间时段。
- 日、月、时用水继续共用趋势图区域，避免同时堆叠多张同类图。
- 使用项目现有 ECharts 依赖，图表支持悬浮提示和窗口自适应。

## 验收重点

- 列表操作只保留“数据分析”。
- 点击后进入对应设备的数据分析页，可返回数据监测列表。
- 日、月、时、夜间用水和报警指标均有数据展示。
- 趋势指标与时间范围切换后图表同步变化。
- 夜间用水首次进入默认本月，重置后也恢复本月。
- 夜间统计摘要和峰谷时刻分布随查询周期同步刷新。
- 夜间用水不出现逐日明细和状态字段。
