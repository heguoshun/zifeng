# 指标计算口径

完整统计口径文档，供后端接口与验收使用。**页面 UI 未必全部展示**——未展示的指标在表中标注「本页不展示」。

### 通用统计前提

| 项目 | 规则 |
| --- | --- |
| 统计范围 | 当前用户数据权限 + 顶部「产品」筛选 |
| 禁用/未启用设备 | 不参与在线率、数据正常率分母 |
| 故障设备 | 计入设备总量；不算在线、不算离线 |
| 环比 | 今日→较昨日；近7天→较前7天；近30天→较前30天 |

### 设备状态分类

| 状态 | 计入在线率分子 | 计入离线分析 | 计入运行异常 |
| --- | --- | --- | --- |
| 在线 | 是 | 否 | 否 |
| 离线 | 否 | 是 | 可联动 |
| 故障 | 否 | 否 | 是 |
| 禁用 | 不参与分母 | 否 | 否 |

### 设备资产类（快照，不受时间范围影响）

设备总量、在用设备、闲置设备、在用率、闲置率。

### 设备在线率类

| 指标 | 本页展示 |
| --- | --- |
| 设备在线率（概况卡片） | 是 |
| 当前在线/离线台数（概况卡片辅助） | 是 |
| 在线率趋势图 | **否** |
| 离线时长分布 | **否**（后端口径预留） |

在线率按时间窗口采样平均，详见日在线率公式。

### 数据正常率类

| 指标 | 本页展示 |
| --- | --- |
| 数据正常率（概况卡片 + 异常区徽标） | 是 |
| 当前异常设备数 | 是 |
| 异常类型分布 | 是 |
| 数据异常趋势条 | 是 |
| 异常次数（累计） | **否**（后端口径） |

### 告警类

| 指标 | 说明 |
| --- | --- |
| 告警总量 / 三态分布 | 统计窗口内触发告警；本页环形图展示 |
| 告警处理率 | 概况卡片展示 |
| 当前待处理告警条数 | 概况卡片摘要「当前待处理告警 N 条」；**当前未处理快照**，与环形图「未处理」分段口径不同 |

### 工单类

工单总量及四态分布按创建时间在统计窗口内汇总；工单结单率本页不单独成卡。

### 健康状态判定

| 指标 | 正常 | 关注 | 异常 |
| --- | --- | --- | --- |
| 设备在线率 | ≥ 98% | 95%～98% | < 95% |
| 数据正常率 | ≥ 99% | 97%～99% | < 97% |
| 告警处理率 | ≥ 95% | 85%～95% | < 85% |
| 整体状态 | 三项均正常 | 任一项关注且无异常 | 任一项异常 |

页头与标签文案：**正常 / 关注 / 异常**（不加修饰词）。

### 待开发前确认参数

在线超时阈值、非整点上报容差、各产品应上报规则、长期离线阈值（用于后端离线时长统计，本页暂不展示）。
