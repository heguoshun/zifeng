# 大表数据分析页决策快照

## 产品需求

- 数据监测分析列表移除“设备详情”和“指标分析”，合并为“数据分析”。
- 点击“数据分析”进入独立页面，展示当前大表的用水和报警分析。
- 核心指标：日用水、月用水、时用水、夜间用水、报警数据。
- 用水数据以趋势图展示，并支持不同指标和时间范围切换。

## 设计决策

- 沿用大表中心现有导航、面包屑、白色面板和按钮样式。
- 顶部展示设备身份、区域、用户、状态和最后数据时间。
- 设备信息下方优先展示近一周逐日明细，包括起止读数、日用水、小时峰值、夜间用水、逆向流量、报警次数和数据状态。
- 明细下方通过页签切换日用水、月用水、时用水、夜间用水和报警数据趋势。
- 所有指标共用一个趋势图区域，避免同时堆叠多张同类图。
- 使用项目现有 ECharts 依赖，图表支持悬浮提示和窗口自适应。

## 验收重点

- 列表操作只保留“数据分析”。
- 点击后进入对应设备的数据分析页，可返回数据监测列表。
- 日、月、时、夜间用水和报警指标均有数据展示。
- 趋势指标与时间范围切换后图表同步变化。
